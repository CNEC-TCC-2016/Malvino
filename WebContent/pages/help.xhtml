* A página inicial será disponibilizado para visualização os veículos dísponiveis, o usuário terá um menu

ao lado esquerdo, onde poderá acessar tais submenus:



-> Home;



-----> Tela onde usuários do sistema visualizarão se tem algum veículo disponível no momento podendo ser utilizado

ou manda-lo para manutenção;
-> Sair;



-----> Sair do sistema.
-> Help;



-----> Auxiliara o usuário nas duvidas frequentes em relação ao sistema.



-> Usuario;



-----> Tela de usuários cadastrados, podendo se realizar tais operações:



* Cadastro de novo usuário;

* Atualizar dados de usuário cadastrados;

* Excluir dados de usuário cadastrados;
-> Veiculo;



-----> Tela de veículos cadastrados, podendo se realizar tais operações:



* Cadastro de novo veículo;

* Atualizar dados de veículo cadastrados;

* Excluir dados de veículo cadastrados;.
-> Rota;



-----> Tela de rotas cadastradas, podendo se realizar tais operações:



* Cadastro de uma nova rota;

* Atualizar dados de rotas cadastradas;

* Excluir dados de rotas cadastradas;.



-> Disponiveis;



-----> Tela de visualizar veículos disponiveis, podendo se realizar tais operações:



* Realizar uma viagem;

* Realizar uma manutenção do veículo;
-> Em uso;



-----> Tela de visualizar veículos em uso, podendo se realizar tais operações:



* Finalizar viagem;
-> Em manutenção;



-----> Tela de visualizar veículos em manutenção, podendo se realizar tais operações:



* Finalizar manutenção do veículo;
